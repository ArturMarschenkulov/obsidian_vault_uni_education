

# Rules
$\int dx$
Note, that + $C$ is always implied.
* $\int x^n dx = \dfrac{1}{n+1}x^{n+1}$
	* $\int x^{an+b} dx = \dfrac{1}{a(n+1)}(ax+b)^{n+1}$
* $\int x^{-1} dx = \ln|x|$
	* $\int (ax+b)^{-1} dx = a^{-1}\ln|ax+b|$
* $\int c^{ax+b} dx = (a\ln(c))^{-1}c^{ax+b}$
* $\int \ln(x) dx=x(\ln(x)-1)$

* $\int f gdx = f G - \int f' G dx$
